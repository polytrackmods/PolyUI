PolyUI

By DoraChad



A library which makes PolyTrack UI elements easier to work with (especially in other mods).


Calling and Using Functions:

In order to call a function from PUI in your mod, you first need to make sure you have PUI listed as a dependency in your mod's manifest.json (check out PML documentation for that).

Next, in order to use it PUI functions in your mod's code, you will have to write this:

let polyUI = pml.getMod("pui");
polyUI.function1();
polyUI.function2();

From here, you can call and use any of the following functions.



All PUI Functions:

addMenuButton:

addMenuButton(image_path, text, onClick, order (optional))

This function adds a button to the main menu screen of polytrack.


Here is an example implementation of this:

polyUI.addMenuButton("images/pending.svg", "Clock", () => {
console.log("Clock Started");
}, 2)

This results in this change:

Notice how the onClick parameter is an inputed as an arrow function so that the console.log does not get executed when addMenuButton is called.
The order of the buttons starts at 1, which represents the left most position (before the customize button).
