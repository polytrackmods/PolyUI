PolyUI

By DoraChad



A library which makes PolyTrack UI elements easier to work with (especially in other mods).


Calling and Using Functions:

In order to call a function from PUI in your mod, you first need to make sure you have PUI listed as a dependency in your mod's manifest.json (check out PML documentation for that).\nNext, in order to use it PUI functions in your mod's code, you will have to do this:\n

let polyUI = pml.getMod("pui"); polyUI.function1(); polyUI.function2();
